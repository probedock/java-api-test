# java-api-test

> A Java library to test JSON APIs with JUnit.

## Usage

1. Put the following dependency in your pom.xml

```xml
<dependency>
  <groupId>io.probedock.test</groupId>
  <artifactId>java-api-test</artifactId>
  <version>1.0.0</version>
</dependency>
```

### Requirements

* Java 6+
